# 前端开发者github的star整理
前端从业几年，积累了不少[github开源库](https://github.com/folger-fan?tab=stars)。
有时候想查阅以前star的库，但不好找，github大多库都是英文说明，对中文开发者不太友好，这里整理下收集的github库，方便需要的时候查阅。
其实老早就有整理github上资源的想法，现在才付诸行动。
正在写的时候收到[稀土圈公众号开源库](https://juejin.im/post/59a7a72b6fb9a0248d24f308)功能上线的通知，英雄所见略同。
